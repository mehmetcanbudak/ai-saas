## Deployment

To deploy this project run

```bash
  go mod init ai-saas
```

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`HTTP_LISTEN_ADDR`

`DB_HOST`
`DB_USER`
`DB_PASSWORD`
`DB_NAME`

`SUPABASE_SECRET`
`SUPABASE_URL`
